2. Which class has the correctly created constructor? Explain why. If they are both correct, explain why.

	public class Example{
		private int shoes;
		private int pants;
		private int shirts;

		public Example(int shoes, int pants, int shirts){
			this.shoes = shoes;
			this.pants = pants;
			this.shirts = shirts;
		}

OR

		public Example(){
			shoes = 5;
			pants = 10;
			shirts = 3;
		}

	}


Answer: 
Both constructors are created correctly, it is an example of overloading. The first constructor contains parameters that you can manipulate in the main method. The second constructor sets initial values to the attributes.

Codingbat problem. Write a method that compares the two given arrays of the same size and 
returns how many characters they have in common at the same position. 
However, the common characters cannot be followed by a 0.

Test case 1. 
input: [0, 5, 6, 2, 1] and [0, 5, 0, 6, 8]

Test case 2.
input: [5, 5, 5, 5, 5] and [5, 5, 5, 5, 5]

Test case 3.
input: [] and []

Test case 4.
input: [1] and [1]

Test case 5.
input: [0, 0, 0, 0, 0] and [0, 0, 0, 0, 0]

Test case 6.
input: [1, 2, 3, 4, 5, 0] and [1, 2, 9, 4, 5, 0]	    
	

public class CodingbatExample {
    private static int compare(int[] nums1, int[] nums2){
        int count = 0;
        for(int i = 0; i < nums1.length; i++){
            if ((i == nums1.length - 1 || (nums1[i+1] != 0 && nums2[i+1] != 0)) && nums1[i] == nums2[i]) {
                count++;
            }
        }
        return count;
    }
}

Solutions: 
Test case 1: 1
Test case 2: 5
Test case 3: 0
Test case 4: 1
Test case 5: 1
Test case 6: 4




Lab Exercise

Steps:
1. Design a class called BankAccount. Declare attributes for hours worked, hourly pay, overtime hours worked, weeks worked, and tip accumulated.
   Hourly pay should not change.
2. Design a constructor that takes in the above attributes as arguments.
3. Design another constructor that represents John’s working conditions. He works as a waiter for 25 weeks. 
   During this time period, an average week consisted of working 30 hours, 5 overtime hours, and 50.00 dollars in tips.
4. Create getter methods for hours, overtime hours, weeks, and tip.
5. Create setter methods for hours, overtime hours, weeks, and tip.
6. Create a method that calculates the yearly salary. Use a for loop to calculate this.
7. Create a method that prints out the yearly salary.
8. Create a main method that calls printSalary() on two new objects, John and mySalary


public class BankAccount{
	private final double MY_PAY = 10.00; //hourly pay
	private int myHours;    //hours worked in a week
	private int myOvertime; //hours worked in overtime
	private int myWeeks;    //weeks worked 
	private double myTip;   //tip money accumulated

	public BankAccount(int hours, int overtime, int weeks, double tip){
		myHours = hours;
		myOvertime = overtime;
		myWeeks = weeks;
		myTip = tip;
	}

	public BankAccount (){
		myHours = 30;
		myOvertime = 10;
		myWeeks = 25;
		myTip = 50.00;
	}

	public int getHours(){
		return myHours;
	}

	public int getOvertime(){
		return myOvertime;
	}	

	public double getTip(){
		return myTip;
	}

	public int getWeeks(){
		return myWeeks;
	}

	public void setHours(int hours){
		myHours = hours;
	}

	public void setOvertime(int overtime){
		myOvertime = overtime;
	}

	public void setWeeks(int weeks){
		myWeeks = weeks;
	}

	public void setTip(double tip){
		myTip = tip;
	}

	public double calcSalary(){
		int salary = 0;
		for(int i = 0; i < myWeeks; i++){
			salary += ((myHours * MY_PAY) + (myOvertime * MY_PAY)) + myTip;
		}
		return salary;
	}

	public void printSalary(){
		System.out.println(“The salary is: “ + calcSalary());
	}


	public static void main(String[] args){
		BankAccount john = new BankAccount();
		BankAccount mySalary = new BankAccount(30, 5, 30, 200.00);
		john.printSalary();
		mySalary.printSalary();
	}
}
